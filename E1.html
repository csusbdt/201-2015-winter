11) Describe the Unix command ls. 

It lists the contents of a directory.

12) Describe the Unix command cd. 

It changes the current directory.

13) Describe the Unix command mkdir. 

It makes a new directory.

14) Describe the Unix command rm. 

It removes (deletes) files.

15) Describe the Unix command pwd. 

It prints (displays) the current working directory.

16) Describe the Unix command mv. 

It moves files and directories.

17) Describe the Unix command rmdir. 

It removes an empty directory.

18) Show the Unix command to change the current directory to your home directory. 

cd ~

19) Suppose you have an empty directory named temp in your home directory. Show the Unix command to remove this directory. 

rmdir ~/temp

20) Suppose you have a directory named temp in your home directory. Suppose there is a file named main.cpp in this temp folder. Show the Unix command to move main.cpp into your home directory. 

mv ~/temp/main.cpp ~/

21) Explain what the following Unix command does.

mv main.cpp .. 

It moves main.cpp into the parent directory of the current directory.

22) What does the dot '.' in the following Unix command mean?

./a.out 

It signifies the current directory.

23) The C++ compiler in Linux generates an executable file named a.out by default. Why do we need to use './' as follows to run the executable from within the directory it is located?

./a.out 

We need to tell the command line interpreter to look 
in the current directory to find the file a.out.
